Kaltura API Documentationmetadata.addFromFile

Allows you to add a metadata object and metadata file associated with Kaltura object

Try it out
post https://www.kaltura.com/api_v3/service/metadata_metadata/action/addFromFile

Parameters

  • body
  • query
xmlFile:file  required

XML metadata

Responses

  • 200
  • Errors

Success
