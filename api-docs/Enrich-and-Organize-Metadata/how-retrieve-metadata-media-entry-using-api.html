How to Retrieve Metadata of a Media Entry using the API?

How to Retrieve Metadata of a Media Entry using the API?

Updated: Tue Aug 8 13:28:22 2017, By Jess Portnoy Contribute to this page

To get a specific video entry, call the API media.get, which provides the ID of the Kaltura Entry.

The media.get request will return the basic metadata and information associated a Kaltura Entry.

 

Using the Kaltura REST API, the media.get request will look as follows.

http://www.kaltura.com/api_v3/index.php?service=media&action=get&entryId={yourentryId}&ks={ks}

Where {yourentryId} is the id of the media entry you’d like to retrieve information about, and {ks} is a valid Kaltura Session of an entitled user who has access to that media entry.

For a full list of the available Media Entry fields that will be returned by the media.get API, visit the media.get API documents.
