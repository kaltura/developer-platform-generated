Parameters

Show
These parameters are valid for multiple operations
ks:string  required
format:integer  required

The format of the response

Allowed values are:
  • 1 (JSON)
  • 2 (XML)
  • 3 (PHP)

captionParams[partnerId]:integer  required
captionParams[name]:string  required

The name of the Flavor Params

captionParams[systemName]:string  required

System name of the Flavor Params

captionParams[description]:string  required

The description of the Flavor Params

captionParams[tags]:string  required

The Flavor Params tags are used to identify the flavor for different usage (e.g. web, hd, mobile)

captionParams[requiredPermissions]:array  required
captionParams[sourceRemoteStorageProfileId]:integer  required

Id of remote storage profile that used to get the source, zero indicates Kaltura data center

captionParams[remoteStorageProfileIds]:integer  required

Comma seperated ids of remote storage profiles that the flavor distributed to, the distribution done by the conversion engine

captionParams[mediaParserType]:string  required

Enum Type: KalturaMediaParserType

Media parser type to be used for post-conversion validation

Allowed values are:
  • 0 (MEDIAINFO)
  • remoteMediaInfo.RemoteMediaInfo (REMOTE_MEDIAINFO)
  • 1 (FFMPEG)

Comma seperated ids of source flavor params this flavor is created from
