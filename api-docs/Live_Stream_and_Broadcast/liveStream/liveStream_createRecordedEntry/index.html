Kaltura API DocumentationliveStream.createRecordedEntry

Live Stream service lets you manage live stream entries

Try it out
get https://www.kaltura.com/api_v3/service/livestream/action/createRecordedEntry

Parameters

  • query
Show
These parameters are valid for multiple operations
ks:string  required
format:integer  required

The format of the response

Allowed values are:
  • 1 (JSON)
  • 2 (XML)
  • 3 (PHP)
entryId:string  required

Live entry id

mediaServerIndex:string  required

Enum Type: KalturaEntryServerNodeType

Media server index primary / secondary

Allowed values are:
  • 0 (LIVE_PRIMARY)
  • 1 (LIVE_BACKUP)
liveEntryStatus:integer  required

Enum Type: KalturaEntryServerNodeStatus

the status KalturaEntryServerNodeStatus::PLAYABLE | KalturaEntryServerNodeStatus::BROADCASTING

Allowed values are:
  • 0 (STOPPED)
  • 1 (PLAYABLE)
  • 2 (BROADCASTING)
  • 3 (AUTHENTICATED)
  • 4 (MARKED_FOR_DELETION)

Responses

  • 200

Success
